# Orbit — VS Code Productivity Tracker

## Links
- **GitHub:** https://github.com/aditya-singh2005/Orbit-Prod-Extension
- **VS Code Marketplace:** https://marketplace.visualstudio.com/items?itemName=orbit-extension.orbit-productivity-tracker

To try it: search **"Orbit - Productivity Tracker"** directly in the VS Code Extensions panel (Ctrl+Shift+X), or install it from the Marketplace link above.

## What it is

A personal productivity co-pilot built as a VS Code extension. Orbit automatically tracks how much time you spend coding, without ever leaving your editor.

## Features

- **Real-time coding tracker** — automatically tracks active coding time in the background
- **Interactive dashboard** — daily stats, weekly insights, streaks, and long-term progress in one view
- **Status bar integration** — always-visible current session time
- **Achievements & streaks** — gamifies consistency with milestones and daily targets
- **Export & refresh** — refresh live stats instantly or export data for personal tracking
- **Persistent storage** — stats are stored in VS Code's global state, so progress survives across sessions; tracks total coding time, daily breakdowns, and live session time

## Commands

Run via Command Palette (Ctrl+Shift+P):
- `Orbit: Show Productivity Stats` — quick summary of total coding time
- `Orbit: Show Productivity Dashboard` — full dashboard with charts, streaks, and achievements

## Tech stack

JavaScript, VS Code Extension API

## Why I built it

I wanted a lightweight way to actually see how much focused coding time I was putting in day to day — treating consistency in coding the same way I treat my morning-run streaks: visible, trackable, and a little gamified so it's easier to stay disciplined about it.
